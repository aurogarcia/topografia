## Fontes de Imagens Sugeridas:

### Gratuitas:
- **Unsplash** (unsplash.com) - Busque por: "surveying", "construction", "civil engineering"
- **Pexels** (pexels.com) - Busque por: "topografia", "construção", "engenharia"

## Dicas Importantes:

1. **Resolução**: Use imagens de alta qualidade (mínimo 1920px de largura)
2. **Formato**: Prefira JPG para fotos, PNG para logos/gráficos
3. **Tamanho**: Otimize as imagens para web (máximo 500KB cada)
4. **Direitos autorais**: Certifique-se de que pode usar as imagens comercialmente

## Como Otimizar Imagens:

1. Use ferramentas online como TinyPNG ou Squoosh
2. Mantenha proporção adequada para cada seção
3. Teste em dispositivos móveis
